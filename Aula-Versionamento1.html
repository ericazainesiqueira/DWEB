Software de Versionamento

É um sistema que tem por objetivo realizar o controle de versões de um código ou documento. Sendo assim é possível gerenciar as versões de um código ou documento realizado por um desenvolvedor.

Como fazer versionamento de software com o GIT ?

O Git é provavelmente a ferramenta de versionamento mais usada em projetos com muitos colaboradores.

Trata-se da plataforma que abriga o GITHUB,serviço online que oferece um servidor git gratuito para projetos de código aberto

É importante conhecer os principais comandos de ferramenta:

*Commit:usado para confirmar as alterações ou enviar uma mensagem.

*Push: trasfere commits do repositorio local

*Fitch:da acesso a dados de projetos remotos

*Pull: incorpora alterações de um repositorio remoto
