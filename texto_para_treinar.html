HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee para publicação de conteúdo em hipertexto, tais como, e que podem ser vinculados através de hyperlinks.

TAGS

As tags definem ao navegador a marcação de conteúdo.
As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao /p
